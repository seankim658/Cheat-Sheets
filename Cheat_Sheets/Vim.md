## Navigating  
### General Navigating 
- `h` move cursor left  
- `j` move cursor down  
- `k` move cursor up  
- `l` move cursor right  
- `<C-U>` half page up  
- `<C-D>` half page down  
- `<C-B>` page up    
- `<C-F>` page down
